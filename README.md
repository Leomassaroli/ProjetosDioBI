## üéì Skills and Knowledge Acquired

### 1. Data Import and Transformation
- **Description:** I learned how to connect Power BI to various data sources such as Excel spreadsheets, SQL databases, and APIs, importing, cleaning, and transforming data using Power Query Editor.
- **Practical Use:** Data transformation ensures the information is clean and structured, allowing for accurate analysis and providing a solid foundation for consistent reporting.

### 2. Data Modeling
- **Description:** I understood the concept of data modeling, which involves creating relationships between tables to organize the data in a logical and efficient manner.
- **Practical Use:** This is essential for building reports and dashboards that correctly reflect the interconnections between data from various areas, such as sales and finance, creating detailed and interactive views.

### 3. Creating Measures and Calculated Columns with DAX
- **Description:** DAX allows you to create custom calculations. I learned how to develop measures and calculated columns using DAX, enabling dynamic and adaptable analysis.
- **Practical Use:** With DAX, it is possible to perform complex calculations such as conditional sums and time-based comparisons, which are critical for insights based on KPIs.

### 4. Data Visualization and Dashboard Design
- **Description:** I developed interactive and intuitive visualizations using bar, column, line, scatter, and map charts, and practiced creating well-organized dashboards to tell a visual story.
- **Practical Use:** The ability to present insights clearly and impactfully facilitates communication with teams and managers, enabling faster and more informed decision-making.

### 5. Publishing and Sharing Reports
- **Description:** I learned how to publish and share reports on Power BI Service, keeping visualizations accessible and collaborative for other team members.
- **Practical Use:** The ability to share insights with stakeholders and teams, using access permissions, ensures data security and maximizes the impact of the analyses.
